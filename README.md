# 📹 Alpha视频批量合成操作指南

## 🎯 写在前面

这个工具是用来批量处理视频的，把你的原始视频和Alpha模板合成在一起，生成新的视频。就像给视频加特效一样简单！

不管你是新手还是老手，跟着这个指南一步步来，保证你能轻松上手。

---

## 🚀 第一步：选择你的素材视频

### 📂 素材选择区域

![素材选择](https://via.placeholder.com/600x200/4CAF50/white?text=素材选择区域)

**这里是干什么的？**
- 这里显示你电脑里所有可以处理的视频文件
- 你可以选择一个或者多个视频来批量处理

**怎么操作？**
1. 在"原素材（可多选）"区域，你会看到所有可用的视频文件
2. 点击视频名字前面的小方框来选择
3. 想全选？点击"✅ 全选"按钮
4. 想清空选择？点击"❌ 清空"按钮

**小贴士：**
- 建议一次不要选太多视频，免得电脑卡顿
- 如果看不到你的视频，检查一下是不是放在了正确的文件夹里

---

## 💾 第二步：管理你的参数预设

### 🎛️ 预设功能区域

![参数预设](https://via.placeholder.com/600x150/2196F3/white?text=参数预设区域)

**什么是预设？**
- 预设就是把你常用的设置保存起来，下次直接用，不用重新设置
- 就像手机的快捷方式一样

**怎么保存预设？**
1. 在"预设名称"框里输入一个好记的名字，比如"高质量设置"或"快速处理"
2. 点击"💾 保存预设"按钮
3. 系统会提示你保存成功

**怎么使用预设？**
1. 点击"选择预设"下拉框
2. 选择你之前保存的预设
3. 点击"📂 加载预设"按钮
4. 所有设置会自动填入

**怎么删除预设？**
1. 选择要删除的预设
2. 点击"🗑️ 删除预设"按钮
3. 确认删除即可

---

## 🎭 第三步：配置Alpha模板

### 🎨 模板选择区域

![Alpha模板配置](https://via.placeholder.com/600x200/FF9800/white?text=Alpha模板配置)

**什么是Alpha模板？**
- Alpha模板就是特效素材，分为三层：顶层、中层、底层
- 就像做三明治一样，一层一层叠起来

**怎么选择模板？**
1. **顶层模板**：选择最上面的特效层
2. **中层模板**：选择中间的特效层
3. **底层模板**：选择最下面的特效层

每个下拉框点击后会显示可用的模板文件，选择你需要的即可。

**小贴士：**
- 不是每层都必须选择，可以只用一层或两层
- 建议先用默认模板试试效果

---

## ⏰ 第四步：时间点控制（高级功能）

### 🎲 随机时间点设置

![时间点控制](https://via.placeholder.com/600x250/9C27B0/white?text=时间点控制)

**这个功能是干什么的？**
- 让视频在不同的时间点开始播放，增加随机性
- 避免所有视频都从同一个时间点开始

**基础随机模式：**
1. 勾选"🎲 启用随机时间点合成"
2. 系统会自动在视频的前面部分随机选择开始时间

**高级随机模式：**
1. 勾选"🔧 启用高级随机时间点模式"
2. 调整"前N秒随机窗口"滑块，设置随机范围（10-120秒）

**精确定点模式：**
1. 勾选"🎯 启用精确定点播放模式"
2. 设置"定点播放时间"，所有视频都从这个时间点开始

**新手建议：**
- 刚开始可以不用这些功能，直接用默认设置
- 熟悉后再尝试随机时间点功能

---

## ✂️ 第五步：分层Alpha模板截取（专业功能）

### 🎬 截取功能设置

![分层截取](https://via.placeholder.com/600x300/607D8B/white?text=分层Alpha模板截取)

**这个功能是干什么的？**
- 可以只使用Alpha模板的一部分，而不是整个模板
- 就像剪辑视频一样，截取你需要的片段

**顶层截取：**
1. 勾选"启用顶层Alpha截取功能"
2. 设置"顶层截取开始时间"（从第几秒开始）
3. 设置"顶层截取持续时间"（截取多长时间）

**中层和底层截取：**
- 操作方法和顶层一样
- 可以分别设置不同的截取时间

**使用建议：**
- 这是专业功能，新手可以跳过
- 如果模板太长，可以用这个功能截取精华部分

---

## ⚙️ 第六步：编码设置

### 🎥 视频质量设置

![编码设置](https://via.placeholder.com/600x200/795548/white?text=编码设置)

**编码预设：**
- 选择预设的编码方案，比如"高质量"、"标准质量"等
- 新手直接选"标准质量"就行

**视频质量 (CRF)：**
- 数值越小，质量越高，文件越大
- 数值越大，质量越低，文件越小
- 推荐值：23（平衡质量和文件大小）

**音频比特率：**
- 控制音频质量
- 192kbps 已经很好了，一般不用调

**小白设置：**
- 编码预设：选"标准质量"
- 视频质量：保持默认的23
- 音频比特率：保持默认的192

## 🔧 第七步：并行处理设置

### ⚡ 性能优化

![并行处理](https://via.placeholder.com/600x150/3F51B5/white?text=并行处理设置)

**最大并行任务数：**
- 同时处理几个视频
- 电脑配置好：可以设置4-6个
- 电脑配置一般：建议设置2-3个
- 电脑配置差：设置1个

**怎么判断电脑配置？**
- 8核以上CPU：可以设置4-6个
- 4-8核CPU：设置2-4个
- 4核以下CPU：设置1-2个

## 🎮 第八步：开始处理

### 🚀 控制面板

![控制面板](https://via.placeholder.com/600x300/4CAF50/white?text=控制面板)

**开始处理：**
1. 确认所有设置都OK了
2. 点击"🚀 开始批量处理"按钮
3. 等待处理完成

**处理过程中：**
- "处理结果"框会显示实时进度
- 可以看到正在处理哪个文件
- 显示处理成功或失败的信息

**停止处理：**
- "⏹️ 停止处理"：正常停止，会完成当前任务
- "🛑 紧急停止"：立即停止，可能丢失当前进度

## 🆘 常见问题解答

### ❓ 遇到问题怎么办？

**Q: 看不到我的视频文件？**
A: 
1. 检查视频格式是否支持（MP4、MOV、AVI等）
2. 确认文件放在了正确的素材文件夹
3. 点击"刷新所有列表"按钮

**Q: 处理速度很慢？**
A:
1. 减少并行任务数
2. 降低视频质量设置
3. 关闭其他占用CPU的程序

**Q: 处理失败了？**
A:
1. 检查原视频是否损坏
2. 确认Alpha模板文件完整
3. 查看"处理结果"框的错误信息

**Q: 输出的视频质量不好？**
A:
1. 降低CRF值（提高质量）
2. 选择更高质量的编码预设
3. 检查原视频质量

**Q: 软件卡死了？**
A:
1. 点击"紧急停止"按钮
2. 重启软件
3. 减少并行任务数再试

## 💡 使用技巧

### 🎯 提高效率的小窍门

**批量处理技巧：**
1. 先用一个视频测试设置，确认效果满意再批量处理
2. 相似的视频可以使用相同的预设
3. 夜间处理大批量视频，避免影响白天工作

**质量优化技巧：**
1. 原视频质量越好，输出质量越好
2. Alpha模板分辨率要匹配原视频
3. 不要过度压缩，保持合理的文件大小

**性能优化技巧：**
1. 关闭不必要的后台程序
2. 确保硬盘空间充足
3. 定期清理临时文件
